In HTML, the <p> (paragraph) tag is used to encapsulate text, providing a structured way to present paragraphs of information. The <p> tag is a block-level element that separates text into distinct paragraphs.

Here's an example of using the <p> tag:

This is a paragraph of text in HTML. It provides structure to the content.

Headings (<h1> to <h6>):

HTML provides six levels of headings, ranging from <h1> as the largest and most important to <h6> as the smallest. Headings are used to define the structure and hierarchy of the content on a webpage.

Example of heading tags:

This is Heading 1

This is Heading 2

This is Heading 6

Line Breaks (<br>):

The <br> tag is used to create a line break within a paragraph, forcing the content that follows to appear on a new line. It is a void or self-closing tag, as it doesn't have a closing tag.

Example of using the <br> tag:

This is the first line.
This is the second line.
