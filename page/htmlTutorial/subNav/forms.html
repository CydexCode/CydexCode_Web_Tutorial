In HTML, forms are created using the <form> tag, which encompasses various form elements for user interaction. Common form elements include text input, password input, checkboxes, radio buttons, and buttons for form submission or resetting.

Examples:

Text Input:

            <form>
<label for="username">Username:</label>
<input type="text" id="username" name="username">
</form>

Password Input:

            <form>
<label for="password">Password:</label>
<input type="password" id="password" name="password">
</form>

Checkbox:

            <form>
<input type="checkbox" id="subscribe" name="subscribe" value="yes">
<label for="subscribe">Subscribe to newsletter</label>
</form>

Radio Buttons:

            <form>
<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label>

<input type="radio" id="female" name="gender" value="female">
<label for="female">Female</label>
</form>

Submit Button:

            <form>
<input type="submit" value="Submit">
</form>

Reset Button:

            <form>
<input type="reset" value="Reset">
</form>
