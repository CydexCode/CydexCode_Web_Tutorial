What is HTML?

HTML, which stands for Hypertext Markup Language, is a standard markup language used to structure content on the web. It provides a set of tags and attributes that define various elements on a webpage.

Basic HTML Elements

Let's look at some basic HTML elements:

  • Headings: Defined using <h1> to <h6> tags.
  • Paragraphs: Created with the <p> tag.
  • Links: Established using the <a> tag with the href attribute.
  • Images: Inserted with the <img> tag and the src attribute.

Example Code

Here's an example illustrating the use of these elements:

                    
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Introduction to HTML</title>
                        </head>
                        <body>
        
                            <header>
                                <h1>Welcome to the World of HTML</h1>
                            </header>
        
                            <section>
                                <h2>What is HTML?</h2>
                                <p>HTML, which stands for Hypertext Markup Language, is a standard markup language used to structure content on the web.</p>
                            </section>
        
                            <section>
                                <h2>Basic HTML Elements</h2>
                                <p>Let's look at some basic HTML elements:</p>
        
                                <ul>
                                    <li><strong>Headings:</strong> Defined using <code>&lt;h1&gt;</code> to <code>&lt;h6&gt;</code> tags.</li>
                                    <li><strong>Paragraphs:</strong> Created with the <code>&lt;p&gt;</code> tag.</li>
                                    <li><strong>Links:</strong> Established using the <code>&lt;a&gt;</code> tag with the <code>href</code> attribute.</li>
                                    <li><strong>Images:</strong> Inserted with the <code>&lt;img&gt;</code> tag and the <code>src</code> attribute.</li>
                                </ul>
                            </section>
        
                        </body>
                        </html>
