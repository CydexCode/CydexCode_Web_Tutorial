HTML Document Structure

Home Next

The structure of an HTML document involves several essential elements. Let's break down the components of the HTML document structure:

  1. <!DOCTYPE html> Declaration: This declaration specifies the HTML version and document type. It informs the web browser about the version of HTML being used, ensuring proper rendering.
  2. <html> Element: The <html> element serves as the root of the HTML document. All other elements are nested within this tag. The lang attribute specifies the language of the document, typically using the ISO language code (e.g., "en" for English).
  3. <head> Section: Within the <html> tag, the <head> section contains meta-information about the document. This includes elements such as the <meta> tag for character set and viewport settings, and the <title> tag to define the title of the document, which appears in the browser's title bar or tab.
  4. <body> Section: The <body> section encapsulates the main content of the HTML document. This is where you include text, images, links, and other elements that make up the visible part of the webpage.

Here's an example illustrating the basic structure of an HTML document: 

                <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Document Structure</title>
  </head>
  <body>
    <!-- Your content goes here -->
  </body>
</html>
